# リリース運用について

Rust Control Station のリリースは、以下の方針で運用します。

- 最新版は、このリポジトリの **GitHub Releases** から配布します。
- 正式リリースに先立ち、動作確認やフィードバック収集を目的としたプレリリース版を公開する場合があります。
- 機能追加、変更、不具合修正などの内容は、各リリースの **Release Notes** を確認してください。
- バージョン番号には [Semantic Versioning](https://semver.org/lang/ja/) を採用します。

## バージョン番号の見方

バージョン番号は `メジャー.マイナー.パッチ`（例：`2.1.3`）の形式です。

- **メジャー**: 互換性のない変更を含む場合に更新します。
- **マイナー**: 後方互換性のある機能追加を含む場合に更新します。
- **パッチ**: 後方互換性のある不具合修正を含む場合に更新します。

本番環境では、各 Release Notes と利用環境の要件を確認してから更新してください。
