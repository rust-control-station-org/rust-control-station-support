# コミュニティガイドライン

誰もが安心して参加できるコミュニティを維持するため、以下のルールを守ってください。

1. **相手を尊重してください。** 他者への誹謗中傷、差別、嫌がらせ、威圧的な言動は禁止します。
2. **投稿前に検索してください。** 同じ質問や提案がないか、Issues と Discussions を確認してください。
3. **不具合は Issues に投稿してください。** 再現可能な不具合は、Issue の Bug Report テンプレートを利用してください。
4. **個人情報を投稿しないでください。** 氏名、住所、メールアドレスなど、ご自身や第三者を特定できる情報は公開しないでください。
5. **秘密情報を共有しないでください。** ログやスクリーンショットには、アクセストークン、API キー、パスワードなどが含まれていないことを確認してください。必要に応じて該当箇所を削除またはマスキングしてください。
6. **建設的に議論してください。** 意見が異なる場合も、人物ではなく内容に焦点を当て、根拠や具体例を添えてください。

ガイドラインに反する投稿は、コミュニティの安全を守るため、運営により編集、非表示、ロックまたは削除される場合があります。
