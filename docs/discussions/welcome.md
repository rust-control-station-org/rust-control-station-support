# Rust Control Station コミュニティへようこそ

このコミュニティは、Rust Control Station の利用者同士が質問、アイデア、利用事例や情報を共有し、より良い製品とコミュニティを一緒につくるための場所です。

## GitHub Discussions の使い方

投稿内容に合うカテゴリを選び、同じ内容の投稿がないか検索してから新しい Discussion を作成してください。質問するときは、利用環境やバージョン、試したことを記載すると回答を得やすくなります。

| 内容 | 投稿先 |
|------|--------|
| 使い方・質問 | **Q&A** |
| 機能要望・改善案 | **Ideas** |
| 不具合報告 | **Issues** |
| 導入事例・スクリーンショット | **Showcase** |
| 運営からの最新情報 | **Announcements** |
| 雑談・情報交換 | **General** |

投稿前に[コミュニティガイドライン](rules.md)と[よくある質問](faq.md)もご確認ください。

皆さまからのフィードバックをお待ちしております。
