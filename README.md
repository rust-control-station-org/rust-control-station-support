# Rust Control Station サポート

Rust Control Station の公式サポートリポジトリです。

このリポジトリでは以下を受け付けています。

- 🐛 不具合報告
- 💡 機能要望
- ❓ 質問・使い方の相談
- 📢 運営からのお知らせ

## コミュニティ

質問や使い方の相談は GitHub Discussions をご利用ください。

| 内容 | 投稿先 |
|------|--------|
| 不具合 | Issues |
| 機能要望 | Discussions（Ideas） |
| 質問 | Discussions（Q&A） |
| 利用事例 | Discussions（Showcase） |
| お知らせ | Discussions（Announcements） |

### Discussion カテゴリ

| カテゴリ | 用途 |

## ご利用方法

### 不具合を報告したい

「Issues」から **Bug Report** を選択し、必要事項を記入してください。

再現手順やスクリーンショットを添付していただけると、調査がスムーズになります。

---

### 新機能を提案したい

Discussions の **Ideas** カテゴリよりご提案ください。

いただいたご意見は今後の開発の参考にさせていただきます。

---

### 質問・相談をしたい

使い方や設定方法などの質問は **Discussions** をご利用ください。

同じ疑問を持つユーザーの参考にもなります。

---

## お願い
